===========================================
      작업 연습 도구 - 프로젝트 정보
===========================================

■ 프로젝트 구조

games/
├── index.html           - 메인 HTML 파일
├── style.css            - 스타일시트
├── game.js             - 메인 게임 로직
├── data/               - 게임 데이터 모듈
│   ├── weapons.js      - 무기 데이터 및 함수
│   ├── attachments.js  - 부착물 데이터 및 계산
│   ├── achievements.js - 업적 시스템
│   └── training.js     - 훈련 프로그램 및 난이도
└── modules/            - 게임 모듈
    ├── gameState.js    - 게임 상태 관리
    ├── renderer.js     - 렌더링 엔진
    └── ui.js          - UI 관리자

■ 주요 기능

1. 레벨 시스템
   - 레벨 1~10
   - 경험치를 획득하여 레벨업
   - 레벨업 시 포인트 보상

2. 무기 시스템
   - 10개의 다양한 무기
   - 레벨에 따라 해금
   - 각 무기마다 고유한 스탯

3. 부착물 시스템
   - 4개 카테고리 (조준경, 총열, 탄창, 손잡이)
   - 총 14개의 부착물
   - 무기 성능 향상

4. 업적 시스템
   - 12개의 업적
   - 업적 달성 시 포인트 보상
   - 진행 상황 추적

5. 훈련 프로그램
   - 추적 훈련
   - 플리킹 훈련
   - 정밀도 훈련
   - 반응속도 훈련
   - 지구력 훈련
   - 종합 훈련

6. 난이도 시스템
   - 쉬움
   - 보통
   - 어려움
   - 극한

7. 게임 모드
   - 모드 1 (클래식): 60초 기본 모드
   - 모드 2 (프리시전): 정밀도 중심
   - 모드 3 (스피드): 빠른 반응 중심

8. 설정
   - 크로스헤어 스타일 변경
   - 크로스헤어 색상 변경

9. 통계 추적
   - 총 게임 플레이 수
   - 총 명중/발사 수
   - 최고 기록
   - 모드별 최고 점수

10. 자동 저장
    - LocalStorage 사용
    - 모든 진행 상황 자동 저장

11. 일시정지
    - ESC 또는 스페이스바로 일시정지
    - 게임 중 언제든 휴식 가능
    - 타이머 자동 정지
    - 타겟 상태 유지

12. 상점 시스템
    - 탄약 구매
    - 특수 아이템 구매
    - 포인트로 결제

13. 특수 타겟
    - 모잠비크 타겟: 몸통 2발 + 머리 1발
    - 고속 타겟: 5초 안에 3발
    - 정밀 타겟: 헤드샷만 3발
    - 부착물 드롭 확률
    - 추가 보상 (포인트, 경험치)

■ 사용 방법

1. 실행
   - index.html을 웹 브라우저로 열기
   - 모든 파일이 같은 폴더에 있어야 함
   - 인터넷 연결 불필요 (오프라인 실행)

2. 조작
   - 마우스 클릭: 사격
   - R 키: 재장전
   - ESC 또는 스페이스바: 일시정지/재개
   - 좌측 패널: 무기/모드 선택
   - 하단 버튼: 각종 기능

3. 게임 진행
   - 난이도 선택
   - 훈련 프로그램 또는 기본 모드 선택
   - 무기 선택
   - 시작 버튼 클릭
   - 타겟 명중으로 점수 획득

■ 기술 스택

- HTML5
- CSS3
- JavaScript (ES6 모듈)
- Canvas API

■ 모듈 시스템

ES6 모듈을 사용하여 코드를 체계적으로 구성:

1. data/ 폴더
   - 정적 게임 데이터
   - 순수 함수로 구성
   - 다른 모듈에서 import하여 사용

2. modules/ 폴더
   - 게임 로직 클래스
   - 상태 관리, 렌더링, UI 등
   - 재사용 가능한 컴포넌트

3. game.js
   - 메인 게임 로직
   - 모든 모듈 통합
   - 이벤트 처리

■ 색상 테마

회사에서 사용하기 적합한 어두운 테마:
- 배경: #0d0d0d, #1a1a1a
- 텍스트: #d4d4d4, #b0b0b0
- 강조: #888, #999
- 테두리: #333, #444

■ 개발자 정보

모든 진행 상황은 브라우저의 LocalStorage에 저장됩니다.
초기화하려면 "초기화" 버튼을 클릭하세요.

■ 주의사항

- 웹 브라우저 필요 (Chrome, Firefox, Edge 등)
- JavaScript 활성화 필요
- 모던 브라우저 권장 (ES6 모듈 지원)
- 파일 구조 변경 시 경로 주의
